# Check-NetLbfoTeamMember

Script à intégrer dans une solution de supervision. Sur Windows, il s'exécute par l'intermédiaire du module NRPE de NSClient++. Le script va vérifier l'état de toutes les équipes (teaming) de cartes réseaux d'un serveur local. S'il y a une carte en erreur dans une équipe, elle va remonter en erreur dans la supervision. Il permet d'identifier une carte défaillante dans une équipe. Suivre le lien ci-dessous pour plus d'informations.
